## Respuestas a preguntas

**1. ¿Qué lista usaría para una Cola y una Pila y por qué?**

- Para una Cola, usaría lista circular doble porque facilita agregar y quitar de los dos lados.
- Para una Pila, usaría lista simple porque solo uso un extremo.

**2. Ventaja y desventaja de crear tu propia lista vs usar la de Java**

- Ventaja: la puedes hacer a tu medida y entenderla bien.
- Desventaja: puede tener errores y tomar más tiempo que usar la lista de Java ya lista y probada.
